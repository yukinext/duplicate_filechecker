# Product Overview

duplicate-filechecker は、ローカルディレクトリ内の重複ファイルを検出し、安全に整理するための CLI ツールです。主な利用者は、動画や大容量ファイルを扱う個人ユーザーで、重複データによるストレージ浪費を抑える目的で利用します。

## Core Capabilities

- ハッシュベース重複検出: ファイル内容からハッシュを算出し、同一内容ファイルを検出する。
- 再帰スキャン: 指定ディレクトリ以下を再帰的に探索し、パターン一致したファイルを対象化する。
- キャッシュ活用: 既存ハッシュを SQLite に保持し、再実行時の再計算を減らして処理時間を短縮する。
- 安全な分離移動: 重複判定後の branch ファイルを trash 領域へ相対パスを保って移動する。
- 保守運用: 実体が消えた DB エントリを定期削除し、監査ログを TSV 形式で残す。

## Target Use Cases

- 収集済みメディア群から重複を探索し、不要コピーを隔離したい。
- 同じ対象ディレクトリを繰り返し検査し、差分中心で高速に運用したい。
- 長期運用で DB と実ファイルの不整合をメンテナンスコマンドで解消したい。

## Value Proposition

- 操作を CLI に集中させた小さく明確な運用フローで、日常メンテナンスに組み込みやすい。
- 「検出」「移動」「保守」を疎結合に分ける設計により、機能追加時の影響範囲を限定しやすい。
- 監査ログと標準ログの併用により、何を削除・移動したかを追跡しやすい。
